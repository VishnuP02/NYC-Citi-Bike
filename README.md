# NYC Citi Bike Analysis

## Overview
This project analyzes New York Cityâ€™s Citi Bike trip data to uncover patterns in ridership behavior, station usage, and seasonal trends. Using Tableau, the analysis focuses on identifying unexpected phenomena that can help inform city planning and operational decisions.

## Objectives
- Explore ridership trends across different times of day and seasons
- Compare behavior between member and casual riders
- Identify high-traffic and low-traffic Citi Bike stations
- Visualize station popularity using interactive maps

## Data Source
- NYC Citi Bike Trip History Data  
- Publicly available datasets provided by the Citi Bike program

## Tools & Technologies
- Tableau (Dashboards, Maps, Stories)
- GitHub (Version control and documentation)

## Key Analyses
- Seasonal and hourly usage patterns
- Differences in trip duration by rider type
- Station popularity and geographic trends
- Interactive city-wide station map with time filters

## Visualizations
The project includes:
- Two interactive Tableau dashboards
- One city-official-style map showing station popularity
- A Tableau Story connecting insights across visuals
